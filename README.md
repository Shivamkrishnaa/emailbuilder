

# emailbuilder

## Requirements

*   Node.js **10+**
## Features

*   Drag and drop components.
*   Edit components properties.
*   Change order of components.
*   Save email template.

### How To Setup?

1. Go to client `cd client`.
2. Run `npm install`.
3. Run `npm run start`.
